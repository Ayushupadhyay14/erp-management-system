
ERP Assignment 

This project is a simple ERP simulation built using Python.

Features:
- CSV processing with error handling
- Inventory & sales management system
- Database integration (Products, Customers, Orders)
- Debugging & optimization example
- Conceptual answers included

Steps to run:
1. Run main.py
2. Database setup optional (configure db_module.py)
